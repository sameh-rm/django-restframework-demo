## üèÅ Getting Started <a name = "getting_started"></a>

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need **Python 3.7** pre installed.

- Clone the repo

  ```
  git clone https://github.com/sameh-rm/django-restframework-demo.git
  ```

### Installing

- create virtual env:
  To create a virtual environment, decide upon a directory where you want to place it, and run the venv module as a script with the directory path:

  ```
  python -m venv env
  ```

* Once you‚Äôve created a virtual environment, you may activate it
  - On Windows, run:
    ```
    tutorial-env\Scripts\activate.bat
    ```
  - On Unix or MacOS, run:
    ```
    source tutorial-env/bin/activate
    ```
* installing packages:
  ```bash
  pip install -r requirements.txt
  ```

- starting the project

  ```
  cd to the backend folder and run the following command
  ```

python manage.py runserver

